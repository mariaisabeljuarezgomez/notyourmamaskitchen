7. The Asset Tray – Built-In & Uploaded Images

Inside the Tools Drawer, at the very top, you will find the Asset Tray — a horizontal scrolling row of image thumbnails. These are your menu's built-in graphic assets (pre-designed borders, decorations, logos) plus any images you have personally uploaded.

Asset Tray — Inside the Drawer
🖼 🖼 🖼 ← Scroll left/right to see all images

Using Built-In Images

The built-in images are pre-loaded decorative graphics designed to match your menu's visual style. To add one to your canvas, simply click or tap its thumbnail in the tray. The image will be placed on the canvas and selected automatically so you can reposition and resize it.

Uploading Your Own Images

  1. Click 🖼 Upload Img in the drawer.
  2. Your device's file picker opens. Select a JPG, PNG, WEBP, GIF, or SVG file.
  3. The image uploads, is stored in your permanent library, appears in the Asset Tray, and is placed on the canvas automatically.
  4. Use the resize handles (visible when selected) to adjust its size, then drag it into position.
Tip: Uploaded images persist in your library permanently. Even if you clear the canvas and start fresh, your uploaded images will still be in the Asset Tray ready to use again.

Transparent Images (PNG with Transparency)

If you upload a PNG file with a transparent background (such as a logo), the editor automatically trims empty transparent space around the image before placing it, so it fits cleanly on your menu without a white box behind it.

Image File Tips

  • Use PNG for logos and graphics with transparent backgrounds
  • Use JPG for photographs of food or your restaurant
  • For best print quality, upload images at 300 DPI or higher resolution
  • File size limit is generous — most phone photos upload without issue

8. Adding & Managing Text Elements

Adding New Text

  1. Open the drawer (🔨 FAB button).
  2. Click ＋ Add Text.
  3. A new element labeled “New Text” appears at the center of your current view, already selected.
  4. Double-click (or double-tap on mobile) to enter editing mode and type your content.
  5. Click anywhere on the canvas background to finish editing.

Selecting an Existing Text Element

Make sure the layout is unlocked (🔓), then click or tap once on any text element. A yellow glowing border appears around it and the Selection Bar shows at the bottom. The element is now selected.

Editing Text Content

With a text element selected, double-click it (or double-tap on mobile) to place your cursor inside. The text becomes editable — you will see a cursor blinking. Type your changes: update a price, fix a name, change a description. Press Enter for a new line. When done, click the canvas background to confirm.

Tip: Text supports multiple lines. Press Enter/Return while editing to create a line break inside the same text element.

Moving Text Elements

  1. Ensure layout is 🔓 Unlocked.
  2. Click the element once to select it (do NOT double-click — that enters text editing mode).
  3. Hold and drag it to its new position.
  4. Release. The new position is recorded in undo history.
  5. Lock the layout again with 🔒 once finished.

Deleting a Text Element

Select the element, then in the Selection Bar click the 🗑 Delete button (red trash icon on the right side of the bar). A confirmation dialog appears. Confirm the deletion. You can restore the element with ↺ Undo Last Change if needed.

Duplicating an Element

Select any element. In the Selection Bar, click the 📋 Duplicate button. An exact copy of the element appears, offset slightly from the original. It is automatically selected so you can immediately move it into position.

Background-Labeled Text Behavior

If a text element is named “background” and set to Locked in the LAYER tab, it will not respond to click selection while the layout is locked. It behaves as a fixed background layer that cannot accidentally be moved or selected. This protects the base design of your menu.

9. Adding & Managing Image Elements

Adding an Image from the Asset Tray

Open the drawer and click any image thumbnail in the Asset Tray. The image is placed on the canvas at a default size and is automatically selected. Drag it to position it, and use the corner resize handles to change its size.

Resizing Images

When an image element is selected, you will see four yellow circular handles at each corner. These are the resize handles.

  • Bottom-right corner: Drag to resize the image freely (width and height independently)
  • Resize is zoom-aware — handles work correctly whether you are zoomed in at 200% or zoomed out at 50%
  • Minimum size protection prevents images from being accidentally shrunk to nothing
Tip: To resize proportionally (keep the image's original shape), hold Shift while dragging a corner handle. The image will scale in proportion automatically.

Moving Images

Same as moving text — ensure layout is 🔓 Unlocked, click the image to select it, then drag it. The center of the image is the drag target. Be sure to grab the image body, not one of the corner handles (those resize, not move).

Image Opacity

Select an image element, go to the LAYER tab in the Selection Bar, and use the Opacity slider to fade the image. Useful for watermarks, decorative backgrounds, or semi-transparent overlays.

Hiding an Image

In the LAYER tab, click the 👁 Visibility toggle to hide the image. It becomes very faint (20% opacity) in the editor so you can still see where it is, but it will not appear in your exported PNG. This is useful for temporarily hiding elements without deleting them.

Locking an Image in Place

In the LAYER tab, click the 🔒 Lock Element toggle. A locked element cannot be moved, resized, or accidentally selected. The lock protects important elements like background frames or watermarks that should never move.

Background images: Any element named “background” and set to Locked will not respond to click selection when the global layout is locked. This means your menu's background image stays completely protected at all times.

10. Adding & Managing Shape Elements

The shape tool lets you add solid-color rectangles to your menu. Use them to create price tag backgrounds, section header banners, colored dividers, highlight boxes, or any decorative block.

Adding a Rectangle

  1. Open the drawer (🔨 FAB).
  2. Click ▢ Add Rect.
  3. A yellow rectangle appears on the canvas, centered in your current view, and is automatically selected.
  4. Use the DESIGN tab in the Selection Bar to customize its appearance.
  5. Drag it to position and use resize handles to adjust size.

Customizing Shape Appearance (DESIGN Tab)

Control What It Does
Fill Color Changes the solid color that fills the rectangle. Click the color swatch to open the color picker.
Corner Radius Rounds the corners of the rectangle. Set to 0 for sharp corners, or increase to create pill shapes and rounded cards.

Opacity Available in the LAYER tab. Fade the shape for watermark or overlay effects.

Layering Shapes Behind Text

To use a rectangle as a background behind text, add the shape first, position and size it, then go to the ARRANGE tab and click Send to Back to move it behind all other elements.

Common Shape Uses

  • Section divider: Thin horizontal rectangle between menu sections

  • Semi-transparent overlay: Low-opacity black rectangle over a photo for text readability

12. The LAYER Tab – Visibility, Opacity, Lock & Role

The LAYER tab appears first in the Selection Bar when you select any element. It controls the element's visibility, whether it can be moved, its transparency, its name, and its role in the menu layout.

LAYER Tab Controls
LAYER | 👁 Visible 🔓 Unlocked Opacity: [========80%===]
Name: [price-header_____] Role: Content ▼

👁 Visibility Toggle

Toggles the element between visible and hidden. Hidden elements show as very faint (🙈 with 20% opacity) in the editor so you can still see their position, but they are completely invisible in the exported PNG. Use this to temporarily remove elements without deleting them.

🔒 Lock Element Toggle

When locked, the element cannot be selected, moved, or resized. The cursor changes when you hover over a locked element to indicate it is protected. Unlock an element anytime by selecting it in the Layers Panel and turning off the lock. This is especially useful for background frames and anchored elements that should never move.

Opacity Slider

Drag the slider left to make the element more transparent, right to make it fully opaque. The value ranges from 0% (completely invisible) to 100% (fully solid). Changes are visible immediately on the canvas and are recorded in undo history.

Element Name

Each element has a name used to identify it in the Layers Panel. You can type a new name in the name field to give it a meaningful label like “main-title” or “price-section-2”. The name does not appear on the menu — it is only used internally for organization.

Role (Content / Background / Overlay)

  • Content — Default. A standard element that participates normally in the design.
  • Background — The element is treated as a background layer. If also Locked, it will not respond to click selection while the global layout is locked, protecting the base design completely.
  • Overlay — The element is treated as a top-layer overlay, always rendered above Content elements in the final export.

13. The DESIGN Tab – Fonts, Color, Size & Spacing

The DESIGN tab contains all styling controls for the selected element. The controls shown depend on the element type — text elements show the full typography suite, shapes show fill and border controls.

DESIGN Tab — Text Element Controls
Font: Century Gothic ▼ Size: 24 B I
Color: Spacing: − 0.05em + Line H: − 1.4 +

Font Family Selector

Click the font dropdown to see all available fonts. The editor includes: Century Gothic, Century Gothic Bold, Century Gothic Bold Italic, Bernard MT Condensed, and Inter. Every font uses font-display: swap for instant display while loading.

Font Size (Numeric Input)

Click the number field next to “Size” and type any number directly. You can enter values like 14, 24.5, 36, or 72. Press Tab or click elsewhere to apply. This precise input lets you set exact sizes without clicking repeatedly.

Bold & Italic

Click B to toggle bold weight. Click I to toggle italic style. Both can be active simultaneously.

Text Color

Click the colored square to open a full color picker. Choose any color using the color wheel, or type a hex code directly (e.g., #95201d for the brand red). The color applies immediately and is saved with your session.

Letter Spacing

Use the − and + buttons to decrease or increase the spacing between individual letters. A value of 0 is normal spacing. Positive values spread letters apart for an elegant headline look.

Line Height

Use the − and + buttons to adjust the spacing between lines of text. A value of 1.4 means line spacing is 1.4× the font size. Increase it for more breathing room in longer descriptions.

Text Alignment

Alignment buttons (Left, Center, Right) control how text is aligned within its element box. Center is common for titles and prices; Left is standard for descriptions.

Shape Design: When a shape is selected in the DESIGN tab, you will see Fill Color, Corner Radius, Border Width, and Border Color instead of typography controls.

14. The ARRANGE Tab – Position, Order & Rotation

The ARRANGE tab gives you precise numerical control over an element's position, size, and layer order. It also includes a rotation control.

ARRANGE Tab Controls
X: 142 Y: 87 W: 280 H: 40
Rotate: [====15°=========] ⇧ Fwd ⇩ Back ⇧⇧ Top ⇩⇩ Bottom

X and Y Position

The X and Y fields show the element's exact position in pixels measured from the top-left corner of the canvas. Type a new number to move the element to an exact coordinate. Useful for aligning multiple elements to the same X value.

Width and Height

Shows and sets the element's exact size in pixels. Type new values to resize precisely. For text elements, the width determines the text wrap boundary.

Rotation

Drag the rotation slider or type a degree value (0–360) to rotate the element around its center point. Useful for angled labels, watermarks, or decorative text at a slant.

Layer Order Controls

  • ⇧ Bring Forward: Moves the element one layer forward, in front of the next element above it
  • ⇩ Send Backward: Moves the element one layer back, behind the next element below it
  • ⇧⇧ Bring to Front: Moves the element to the very front, on top of everything else
  • ⇩⇩ Send to Back: Moves the element to the very back, behind everything else
Common use: When you add a rectangle behind text, click the rectangle, go to ARRANGE → Send to Back. The text always appears on top of the colored background shape.

15. The Layers Panel

The Layers Panel is located at the bottom of the Tools Drawer. It shows a complete list of every element on your canvas so you can see, select, hide, lock, and organize them all from one place.

Layers Panel — Inside the Drawer

Reading the Layers Panel

  • Active indicator (★/○): The currently selected element is highlighted with a gold border and a star
  • Element name: The label you assigned in the LAYER tab
  • Type badge (T/I/S): T = Text, I = Image, S = Shape
  • Visibility icon (👁/🙈): Click to toggle visible or hidden
  • Lock icon (🔓/🔒): Click to lock or unlock the element
  • Strikethrough name: Indicates the element is currently hidden

Selecting Elements from the Layers Panel

Click any element's row in the Layers Panel to select that element on the canvas. This is especially helpful for selecting elements that are hidden, stacked under other elements, or too small to click precisely on the canvas.

Toggling Visibility and Lock from the Panel

Click the visibility or lock icon in any row to instantly change that element's state without selecting it first. This lets you quickly manage multiple elements in a row.

17. Save, Load & Auto-Save

💾 Save Session

Saves your entire current state to the server. This includes every element's content, position, style, layer settings, visibility, lock state, and the positions of all custom images. You will see a green toast: “Project Saved Locally”.

📁 Load Session

Loads the latest server-saved version of your menu. Use this when continuing on a new device or browser. All unsaved local changes will be replaced by the server version. Always save before loading if you have unsaved work you want to keep.

Auto-Save on Export

Every time you click ⇩ Export Pro PNG, the editor automatically runs a Save first. This ensures your exported file always matches exactly what is saved on the server.

Local Browser Backup

In addition to the server save, the editor maintains a local backup in your browser's storage. This protects your most recent work even if your internet connection drops temporarily.

Best practice: Save after completing each menu section, before closing the browser, and before switching to another device. These three habits protect all your work.
