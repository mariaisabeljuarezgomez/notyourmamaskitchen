1. Bienvenida

Bienvenido a Dine In Menu Editor Pro.

Este editor le permite actualizar el texto del menú de su restaurante, mover elementos cuando sea necesario, guardar su progreso, continuar en otro dispositivo y exportar un archivo profesional listo para impresión.

Está diseñado para ser comprensible en el uso diario del negocio, sin requerir software de diseño ni conocimientos técnicos avanzados.

Qué Incluye Este Manual

  • Cómo editar texto y mover elementos de forma segura
  • Cómo funcionan el guardado y la carga entre dispositivos
  • Cómo funciona la versión móvil, incluyendo desplazamiento de la barra, paneo, pinch para zoom y controles táctiles seguros
  • Cómo exportar un PNG final profesional

2. Antes de Empezar

Navegador Recomendado

  • Google Chrome
  • Microsoft Edge
  • Apple Safari

Conexión a Internet

Se requiere una conexión activa a internet para el guardado permanente y la sincronización entre dispositivos. Save Session guarda su trabajo en el servidor y Load Session recupera la última versión guardada cuando continúa en otro dispositivo. [file:1]

Calidad de Vista Previa

Mientras edita, el fondo puede mostrarse con menor resolución para mantener el editor rápido y responsivo. La exportación final siempre usa la fuente original de alta resolución. [file:1]

Mejor hábito: Guarde con frecuencia, especialmente después de cambios importantes o antes de cambiar de dispositivo.

3. Flujo de Inicio Rápido

  1. Abra el editor usando el enlace proporcionado.
  2. Abra el menú principal tocando o haciendo clic en el botón flotante (FAB).
  3. Active Edit Mode.
  4. Mantenga Layout Locked activado mientras edita texto.
  5. Seleccione un elemento de texto y haga sus cambios.
  6. Si necesita mover un elemento, cambie a Layout Unlocked, reposiciónelo y vuelva inmediatamente a Layout Locked. [file:1]
  7. Haga clic en Save Session para guardar su trabajo en el servidor. [file:1]
  8. Haga clic en Export Pro PNG cuando esté listo para crear el archivo final listo para impresión. [file:1]

4. Qué Afecta Cada Acción

Borrador Local

Mientras trabaja, el editor conserva un borrador temporal en su navegador actual. Esto ayuda a proteger cambios recientes si la página se cierra inesperadamente, pero no es su versión guardada de forma permanente.

Save Session

Save Session sube su trabajo actual al sistema permanente respaldado por servidor para que pueda recuperarse después o cargarse en otro dispositivo. [file:1]

Load Session

Load Session reemplaza su vista actual con la última versión guardada en el servidor. [file:1]

Undo Last Change

Undo Last Change revierte una acción reciente a la vez, hasta 50 pasos, y cubre ediciones de texto, movimientos, cambios de estilo, agregar texto y eliminar elementos. [file:1]

Reset to Original

Reset to Original restaura la plantilla original y elimina todas las ediciones personalizadas. Es independiente de Undo Last Change y debe usarse solo cuando desea reiniciar por completo. [file:1]

Export Pro PNG

Export Pro PNG crea el archivo PNG final de 12 pulgadas y 300 DPI para impresión profesional o entrega final. [file:1]

5. Referencia de Controles Principales

Botón Flotante (FAB)

El botón flotante es el punto principal de entrada al panel de controles. Úselo para acceder al modo de edición, bloqueo de diseño, guardar, cargar, deshacer, agregar texto, exportar y reiniciar.

Edit Mode

Active Edit Mode cuando quiera cambiar texto como precios, nombres de platillos, descripciones o títulos.

Layout Locked / Layout Unlocked

Layout Locked es el estado predeterminado cada vez que el editor se abre. Úselo para editar texto, hacer paneo, zoom y navegar de forma segura. Use Layout Unlocked solo cuando quiera mover físicamente un elemento a otra posición. [file:1]

Save Session

Use Save Session después de cambios importantes, antes de cerrar el editor y antes de cambiar de dispositivo. [file:1]

Load Session

Use Load Session para recuperar la última versión guardada en el servidor al continuar en otro dispositivo. [file:1]

Undo Last Change

Undo Last Change revierte una acción a la vez y cubre ediciones de texto, elementos movidos, cambios de estilo, texto agregado y elementos eliminados. [file:1]

Add Text

Add Text crea un nuevo elemento de texto en blanco en el centro de su vista actual. [file:1]

Export Pro PNG

Export Pro PNG crea el PNG final de alta resolución usando la fuente original de alta resolución en lugar de la vista previa de edición. [file:1]

Reset to Original

Reset to Original elimina todos los cambios personalizados y restaura la plantilla original. Esta acción es independiente de Undo y no puede revertirse con Undo Last Change. [file:1]

6. Barra de Herramientas de Selección

La barra de herramientas de selección aparece cuando se selecciona un elemento de texto.

  • Font Dropdown para cambiar la fuente
  • Size (+ / -) para cambiar el tamaño del texto
  • Color Picker para cambiar el color del texto
  • Letter Spacing para ajustar el espacio entre letras
  • Delete Element para eliminar el elemento seleccionado

Comportamiento en Móvil

En dispositivos móviles, la barra aparece cerca de la parte inferior de la pantalla y puede ser desplazable horizontalmente porque la pantalla del teléfono es más estrecha. Si no ve todas las herramientas de inmediato, deslice hacia la izquierda o derecha sobre la barra para revelar las opciones restantes.

Eliminar un elemento requiere confirmación, y el elemento puede recuperarse con Undo Last Change siempre que todavía esté dentro del historial de deshacer. [file:1]

7. Modo de Edición vs. Bloqueo de Diseño

FunciónControlaMejor Para
Edit ModeContenido como palabras y preciosCambiar texto, corregir errores y actualizar precios
Layout LockPosición en la páginaMover bloques de texto, encabezados o alineación de precios

Use Edit Mode cuando quiera cambiar palabras. Use Layout Unlocked solo cuando quiera mover algo. Regrese a Layout Locked tan pronto como el elemento quede en su lugar. [file:1]

Esta separación es especialmente importante en móvil, donde los gestos táctiles podrían causar movimiento accidental. [file:1]

8. Comportamiento Detallado de Edición

Seleccionar y Editar Texto

  1. Asegúrese de que Edit Mode esté activado.
  2. Toque o haga clic una vez sobre un elemento para seleccionarlo.
  3. Aparecerán un recuadro de selección y la barra de herramientas.
  4. Toque el texto otra vez, o haga doble toque/doble clic según su dispositivo, para colocar el cursor.
  5. Escriba sus cambios.
  6. Toque el fondo o seleccione otro elemento para terminar la edición.

Mover Elementos

  1. Abra el menú principal.
  2. Cambie a Layout Unlocked.
  3. Seleccione el elemento que desea mover.
  4. Arrástrelo a la nueva posición.
  5. Regrese a Layout Locked inmediatamente después de colocarlo. [file:1]

9. Sincronización entre Dispositivos

  1. Termine sus ediciones en el primer dispositivo.
  2. Haga clic en Save Session.
  3. Abra el mismo enlace del editor en el segundo dispositivo.
  4. Haga clic en Load Session.

Si olvida guardar en el primer dispositivo, el segundo cargará la última versión guardada, no sus cambios más recientes sin guardar. [file:1]

10. Guía Móvil

La versión móvil está diseñada para una edición segura en teléfonos y tabletas. Como la interacción táctil es menos precisa que el mouse, el flujo protege primero el diseño y solo permite mover elementos cuando usted lo desbloquea de forma intencional. [file:1]

Cómo Funciona la Versión Móvil

El diseño se abre bloqueado por defecto para que pueda hacer paneo, zoom y revisar el menú sin arrastrar texto accidentalmente. [file:1]

Áreas Principales en Móvil

  • FAB: Abre el menú principal con Edit Mode, Layout Lock, Save Session, Load Session, Undo Last Change, Add Text, Export Pro PNG y Reset to Original.
  • Área del Lienzo: La zona visual principal donde hace paneo, zoom, selecciona texto y revisa el espaciado.
  • Barra Inferior de Selección: Aparece después de seleccionar texto y puede ser desplazable horizontalmente en teléfonos.
  • Controles Flotantes de Zoom: Cuando Layout Unlocked está activo, deben aparecer controles flotantes + y - para un control más preciso. [file:1]

Gestos Móviles

  • Paneo: Use un dedo para moverse por el lienzo.
  • Pinch para Zoom: Use dos dedos para acercar o alejar.
  • Seleccionar: Toque una vez para seleccionar un elemento de texto.
  • Editar: Toque nuevamente después de seleccionar para colocar el cursor y mostrar el teclado.
  • Arrastrar para Mover: Solo mientras Layout Unlocked esté activado.

Protección Multi-Touch

Si comienza un gesto multitáctil mientras está arrastrando, el arrastre debe cancelarse o suprimirse inmediatamente para que el diseño no se desplace accidentalmente durante el pinch zoom. [file:1]

Mejor flujo en móvil: Mantenga Layout Locked activado mientras navega y edita texto. Desbloquee solo cuando vaya a reposicionar algo y vuelva a bloquear inmediatamente después. [file:1]

11. Solución de Problemas

Puedo escribir texto pero no puedo mover elementos

Asegúrese de que Layout Unlocked esté activado. El editor se abre en Layout Locked por defecto. [file:1]

Puedo mover elementos pero no puedo editar texto

Asegúrese de que Edit Mode esté activado.

No puedo ver todos los botones de la barra en mi teléfono

Deslice a la izquierda o derecha sobre la barra inferior. En móvil, puede ser desplazable horizontalmente para que todas las herramientas quepan en pantalla.

Mis cambios no aparecen en otro dispositivo

Asegúrese de haber hecho clic en Save Session en el primer dispositivo antes de usar Load Session en el segundo. [file:1]

La vista previa del editor se ve de menor calidad

Esto es normal. El editor usa una vista previa ligera para mayor velocidad, mientras que Export Pro PNG usa la fuente original de alta resolución para el resultado final. [file:1]

12. Buenas Prácticas

  • Mantenga Layout Locked activado a menos que esté moviendo elementos activamente.
  • Guarde después de cada sección importante del trabajo.
  • Use Undo Last Change inmediatamente cuando note un error. [file:1]
  • Revise su PNG exportado antes de enviarlo a impresión.
  • En móvil, haga zoom antes de seleccionar precios pequeños o texto fino.
  • Deslice la barra en móvil si no todas las herramientas están visibles.
  • Vuelva a bloquear el diseño inmediatamente después de mover cualquier elemento. [file:1]

13. Especificaciones Técnicas

Esta sección se incluye como un resumen técnico profesional del estado de producción aceptado. Ayuda a documentar los estándares de calidad y confiabilidad detrás del editor, incluso para lectores que no son técnicos. [file:1]

Estándar de Exportación

La exportación final permanece como un PNG determinístico de alta resolución de 12 pulgadas de ancho y 300 DPI. [file:1]

Fuente de Exportación

La vista previa del editor está optimizada para velocidad, mientras que la exportación usa el fondo original de alta resolución. [file:1]

Persistencia

El guardado permanente está respaldado por servidor, con persistencia en volumen de Railway aceptada como base. [file:1]

Flujo Entre Dispositivos

La persistencia entre dispositivos y navegadores fue verificada como parte del estado base aceptado. [file:1]

Sistema de Undo

Undo Last Change usa un historial real paso a paso que cubre ediciones de texto, movimientos, cambios de estilo, agregar texto y eliminar. [file:1]

Seguridad de Reinicio

Reset to Original permanece separado de Undo Last Change y se conserva como una acción deliberada de restauración. [file:1]

Interacción Móvil

El diseño se abre bloqueado por defecto en móvil, el arrastre se cancela durante multi-touch y aparecen controles flotantes + / - mientras está desbloqueado. [file:1]

Rendimiento

El tiempo de carga inicial se optimizó al sacar el fondo pesado de la ruta crítica del editor y mantener una vista previa ligera para edición. [file:1]

Sistema de Notificaciones

El editor usa un sistema de modales y toasts con marca propia en lugar de alertas nativas del navegador. [file:1]

Estilo de Marca

Los colores aprobados de notificación incluyen rojo profundo #95201d, amarillo #f8f4ad, negro y blanco. [file:1]

Nota profesional: Estas especificaciones reflejan la base de producción estabilizada y ayudan a demostrar que el editor está construido sobre estándares deliberados de calidad, seguridad y salida profesional. [file:1]
