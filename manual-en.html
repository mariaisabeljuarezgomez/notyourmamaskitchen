1. Welcome & What Is New in V2

Welcome to Menu Editor Pro V2 — the most advanced version of the professional menu editing system created by MARIAS DIGITAL DESIGNS.

This editor runs entirely in your web browser. There is nothing to install. You simply open your personal editor link, make your changes, save, and export a print-ready file. It works on desktop computers, laptops, tablets, and phones.

What This Manual Covers

  • Every button, control, and feature in the editor explained in plain language
  • How to add, edit, style, move, and delete text, images, and shapes
  • How the layer system keeps your menu organized
  • How to save, load, and sync your work across devices
  • How to export a professional 12×18 inch 300 DPI print-ready PNG
  • Full mobile workflow including touch gestures and pinch zoom
  • Keyboard shortcuts for faster editing
  • Troubleshooting every common issue

What Is New in Version 2

Layer System

Every element now has a layer role (Content, Background, Overlay), visibility toggle, lock toggle, and opacity control managed through a dedicated Layers Panel.

3-Tab Selection Bar

When you select any element, a floating toolbar appears with three tabs: Layer, Design, and Arrange — giving you full control over every property without opening any menus.

Image Upload Library

Upload your own photos and logos directly from your device. They are saved permanently to your library and accessible from the Asset Tray at any time.

Shape Tool

Add colored rectangles with custom fill color, corner radius, border width, and border color to create price tags, banners, dividers, or decorative blocks.

Font Size Control

Directly type a precise font size number for any text element. No more plus/minus stepping — type exactly 17.5 or 33 or any value you need.

Auto-Save on Export

Every time you click Export, the editor automatically saves your session first so your export always reflects the latest saved state.

Zoom-Aware Resizing

Image resize handles now work correctly at any zoom level. Dragging a corner always resizes by the correct amount regardless of how zoomed in you are.

Keyboard Shortcuts

Ctrl+Z for Undo and Ctrl+E for Export are now available for faster professional workflow on desktop.

2. Before You Start

Recommended Browsers

  • Google Chrome — Best overall experience on desktop and Android
  • Microsoft Edge — Excellent on Windows desktop
  • Apple Safari — Best on iPhone and iPad
  • Mozilla Firefox — Fully supported on desktop

Internet Connection

An active internet connection is required for saving to the server and loading previously saved work on another device. The editor will also use your browser's local storage as a backup so your most recent work is protected even if the connection drops temporarily.

Screen Size & Orientation

The editor works on any screen size. On desktop, you will see the full canvas and all controls simultaneously. On phones, the canvas fills the screen and controls appear as a floating toolbar. If you are on a phone or tablet, landscape orientation gives you more working space and makes it easier to see the toolbar controls.

Preview Quality vs. Export Quality

While you are editing, the background image on the canvas is displayed at a lower resolution so the editor loads quickly and stays responsive. This is normal and intentional. When you click Export Pro PNG, the editor automatically uses the original full-resolution background file, which produces a crisp, professional 300 DPI print file regardless of how the preview looked.

Best habit: Save your session regularly — especially after completing a section of work, before closing the browser, and before switching to another device. One click on Save protects everything.

Your Editor Link

Your editor is accessed through a private web address (URL) provided to you by MARIAS DIGITAL DESIGNS. Bookmark it in your browser so you can return to it quickly. Never share this link publicly — it is your private editing environment.

Tip: On iPhone or Android, you can save your editor link to your home screen so it opens like an app. In Safari or Chrome, use the browser menu and select “Add to Home Screen.”

4. Quick Start Workflow

If you are new to the editor, follow these steps for your first editing session. Each step is explained in full detail in later sections of this manual.

1

Open your editor link in Google Chrome or Safari. The menu loads automatically with your last saved session.

2

Unlock the layout by clicking the 🔒 Layout Locked button in the top header. It will turn yellow and say 🔓 Layout Unlocked. This allows you to click and select elements.

3

Click any text element on the canvas to select it. A yellow highlight box will appear around it, and the Selection Bar will appear at the bottom of the screen.

4

Double-click the text (or double-tap on mobile) to place your cursor inside it. Type your changes — update a price, fix a spelling, change a dish name.

5

Click the canvas background (not on any element) to deselect and finish editing that text.

6

Use the DESIGN tab in the Selection Bar to change the font, color, size, line height, or letter spacing of the selected text.

7

Click 💾 Save in the top header bar to save your work to the server. You will see a green toast notification confirm the save.

8

When your menu is complete, open the drawer (🔨 FAB button) and click ⇩ Export Pro PNG. Your print-ready 300 DPI file will download automatically.

Remember: The Layout Lock button controls whether elements can be moved by dragging. Keep it Locked whenever you are just editing text or browsing the menu. Unlock it only when you need to physically drag something to a new position, then lock it again immediately.

5. The Top Header Bar

The top header bar is always visible at the top of the screen. It contains four controls that you will use constantly throughout every editing session.

Top Header — Full View
MENU EDITOR PRO V2 💾 Save ↺ Undo Last Change 🔒 Layout Locked ↺ RELOAD

💾 Save

Clicking Save immediately saves your entire current session — all text content, positions, styles, layer settings, and image placements — to the server. It also creates a local browser backup at the same time. A green toast message will confirm: “Project Saved Locally”.

Use Save frequently. There is no automatic timed saving — only you control when your work is saved.

Shortcut: Save is also triggered automatically every time you click Export Pro PNG, so your export always matches your most recent work.

↺ Undo Last Change

Clicking Undo Last Change reverses your most recent action one step at a time. The undo system remembers up to 30 actions, including text edits, element moves, style changes, element additions, deletions, visibility changes, and lock toggles. You can click it repeatedly to step back through your history.

Important: Undo does NOT reverse a Reset to Original or a Load Session. Those are intentional full-restore actions.

🔒 Layout Locked / 🔓 Layout Unlocked

This is one of the most important controls in the editor. It has two states:

  • 🔒 Layout Locked (default, red button): Elements cannot be accidentally dragged or moved. Use this state for all text editing, styling, and general navigation. The canvas is safe to touch and scroll.
  • 🔓 Layout Unlocked (yellow button): Elements CAN be dragged to new positions. The floating Zoom Controls (+/−) also appear in the top-right corner when unlocked. Use this only when you intentionally need to reposition something, then switch back to Locked immediately.
Rule of thumb: If you are not actively dragging something, the layout should be Locked. This prevents accidental repositioning, especially on mobile where touch is less precise.

↺ RELOAD

The RELOAD button refreshes the entire page. It is a hard browser reload that resets the editor session. Use it only if the editor becomes unresponsive or something appears visually broken. Save your work before clicking RELOAD — any unsaved changes will be lost.

💾 Save Session

Same as the Save button in the header bar. Saves all your current work to the server permanently. Also backs up to local browser storage. Always use this before closing the editor or switching devices.

📁 Load Session

Loads the last version you saved to the server. Use this when you open the editor on a different device and want to continue where you left off. Warning: This will replace your current unsaved work with the server version. Save first if you have unsaved changes you want to keep.

＋ Add Text

Creates a brand new text element in the center of your current view. The new text will say “New Text” and will be selected automatically so you can immediately type your content, change the font, size, and color from the Selection Bar.

▢ Add Rect

Creates a new yellow rectangle shape in the center of the canvas. Use rectangles to create price tag backgrounds, section dividers, colored banners, or any decorative block. After adding, use the DESIGN tab in the Selection Bar to change the fill color, corner roundness, border width, and border color.

🖼 Upload Img

Opens your device's file picker so you can upload a photo, logo, or graphic from your device. Supported formats include JPG, PNG, WEBP, GIF, and SVG. The image is automatically saved to your permanent image library (you will see it appear in the Asset Tray) and added to your canvas. Transparent PNG files have their transparent borders automatically trimmed so the image fits cleanly.

👁 Toggle Original BG

Shows or hides the original scanned menu background image. Hiding it lets you see only the text and elements you have added, which is useful for checking alignment. The background visibility setting is saved with your session.

⇩ Export Pro PNG

Generates a professional, print-ready PNG file at 12 inches × 18 inches at 300 DPI (3600 × 5400 pixels). The editor automatically saves your session first, then renders every visible element at full resolution, and downloads the file to your device. This is covered in full detail in Section 19.

↻ Reset to Original

Restores the menu to its original template state, clearing all your custom edits. A confirmation dialog will appear asking you to confirm before the reset happens. This cannot be undone with the Undo button. Use it only when you want a complete fresh start.

7. The Asset Tray – Built-In & Uploaded Images

Inside the Tools Drawer, at the very top, you will find the Asset Tray — a horizontal scrolling row of image thumbnails. These are your menu's built-in graphic assets (pre-designed borders, decorations, logos) plus any images you have personally uploaded.

Asset Tray — Inside the Drawer
🖼 🖼 🖼 ← Scroll left/right to see all images

Using Built-In Images

The built-in images are pre-loaded decorative graphics designed to match your menu's visual style. To add one to your canvas, simply click or tap its thumbnail in the tray. The image will be placed on the canvas and selected automatically so you can reposition and resize it.

Uploading Your Own Images

  1. Click 🖼 Upload Img in the drawer.
  2. Your device's file picker opens. Select a JPG, PNG, WEBP, GIF, or SVG file.
  3. The image uploads, is stored in your permanent library, appears in the Asset Tray, and is placed on the canvas automatically.
  4. Use the resize handles (visible when selected) to adjust its size, then drag it into position.
Tip: Uploaded images persist in your library permanently. Even if you clear the canvas and start fresh, your uploaded images will still be in the Asset Tray ready to use again.

Transparent Images (PNG with Transparency)

If you upload a PNG file with a transparent background (such as a logo), the editor automatically trims empty transparent space around the image before placing it, so it fits cleanly on your menu without a white box behind it.

Image File Tips

  • Use PNG for logos and graphics with transparent backgrounds
  • Use JPG for photographs of food or your restaurant
  • For best print quality, upload images at 300 DPI or higher resolution
  • File size limit is generous — most phone photos upload without issue

8. Adding & Managing Text Elements

Adding New Text

  1. Open the drawer (🔨 FAB button).
  2. Click ＋ Add Text.
  3. A new element labeled “New Text” appears at the center of your current view, already selected.
  4. Double-click (or double-tap on mobile) to enter editing mode and type your content.
  5. Click anywhere on the canvas background to finish editing.

Selecting an Existing Text Element

Make sure the layout is unlocked (🔓), then click or tap once on any text element. A yellow glowing border appears around it and the Selection Bar shows at the bottom. The element is now selected.

Editing Text Content

With a text element selected, double-click it (or double-tap on mobile) to place your cursor inside. The text becomes editable — you will see a cursor blinking. Type your changes: update a price, fix a name, change a description. Press Enter for a new line. When done, click the canvas background to confirm.

Tip: Text supports multiple lines. Press Enter/Return while editing to create a line break inside the same text element.

Moving Text Elements

  1. Ensure layout is 🔓 Unlocked.
  2. Click the element once to select it (do NOT double-click — that enters text editing mode).
  3. Hold and drag it to its new position.
  4. Release. The new position is recorded in undo history.
  5. Lock the layout again with 🔒 once finished.

Deleting a Text Element

Select the element, then in the Selection Bar click the 🗑 Delete button (red trash icon on the right side of the bar). A confirmation dialog appears. Confirm the deletion. You can restore the element with ↺ Undo Last Change if needed.

Duplicating an Element

Select any element. In the Selection Bar, click the 📋 Duplicate button. An exact copy of the element appears, offset slightly from the original. It is automatically selected so you can immediately move it into position.

Background-Labeled Text Behavior

If a text element is named “background” and set to Locked in the LAYER tab, it will not respond to click selection while the layout is locked. It behaves as a fixed background layer that cannot accidentally be moved or selected. This protects the base design of your menu.

9. Adding & Managing Image Elements

Adding an Image from the Asset Tray

Open the drawer and click any image thumbnail in the Asset Tray. The image is placed on the canvas at a default size and is automatically selected. Drag it to position it, and use the corner resize handles to change its size.

Resizing Images

When an image element is selected, you will see four yellow circular handles at each corner. These are the resize handles.

  • Bottom-right corner: Drag to resize the image freely (width and height independently)
  • Resize is zoom-aware — handles work correctly whether you are zoomed in at 200% or zoomed out at 50%
  • Minimum size protection prevents images from being accidentally shrunk to nothing
Tip: To resize proportionally (keep the image's original shape), hold Shift while dragging a corner handle. The image will scale in proportion automatically.

Moving Images

Same as moving text — ensure layout is 🔓 Unlocked, click the image to select it, then drag it. The center of the image is the drag target. Be sure to grab the image body, not one of the corner handles (those resize, not move).

Image Opacity

Select an image element, go to the LAYER tab in the Selection Bar, and use the Opacity slider to fade the image. Useful for watermarks, decorative backgrounds, or semi-transparent overlays.

Hiding an Image

In the LAYER tab, click the 👁 Visibility toggle to hide the image. It becomes very faint (20% opacity) in the editor so you can still see where it is, but it will not appear in your exported PNG. This is useful for temporarily hiding elements without deleting them.

Locking an Image in Place

In the LAYER tab, click the 🔒 Lock Element toggle. A locked element cannot be moved, resized, or accidentally selected. The lock protects important elements like background frames or watermarks that should never move.

Background images: Any element named “background” and set to Locked will not respond to click selection when the global layout is locked. This means your menu's background image stays completely protected at all times.

12. The LAYER Tab – Visibility, Opacity, Lock & Role

The LAYER tab appears first in the Selection Bar when you select any element. It controls the element's visibility, whether it can be moved, its transparency, its name, and its role in the menu layout.

LAYER Tab Controls
LAYER | 👁 Visible 🔓 Unlocked Opacity: [========80%===]
Name: [price-header_____] Role: Content ▼

👁 Visibility Toggle

Toggles the element between visible and hidden. Hidden elements show as very faint (🙈 with 20% opacity) in the editor so you can still see their position, but they are completely invisible in the exported PNG. Use this to temporarily remove elements without deleting them.

🔒 Lock Element Toggle

When locked, the element cannot be selected, moved, or resized. The cursor changes when you hover over a locked element to indicate it is protected. Unlock an element anytime by selecting it in the Layers Panel and turning off the lock. This is especially useful for background frames and anchored elements that should never move.

Opacity Slider

Drag the slider left to make the element more transparent, right to make it fully opaque. The value ranges from 0% (completely invisible) to 100% (fully solid). Changes are visible immediately on the canvas and are recorded in undo history.

Element Name

Each element has a name used to identify it in the Layers Panel. You can type a new name in the name field to give it a meaningful label like “main-title” or “price-section-2”. The name does not appear on the menu — it is only used internally for organization.

Role (Content / Background / Overlay)

  • Content — Default. A standard element that participates normally in the design.
  • Background — The element is treated as a background layer. If also Locked, it will not respond to click selection while the global layout is locked, protecting the base design completely.
  • Overlay — The element is treated as a top-layer overlay, always rendered above Content elements in the final export.

13. The DESIGN Tab – Fonts, Color, Size & Spacing

The DESIGN tab contains all styling controls for the selected element. The controls shown depend on the element type — text elements show the full typography suite, shapes show fill and border controls.

DESIGN Tab — Text Element Controls
Font: Century Gothic ▼ Size: 24 B I
Color: Spacing: − 0.05em + Line H: − 1.4 +

Font Family Selector

Click the font dropdown to see all available fonts. The editor includes: Century Gothic, Century Gothic Bold, Century Gothic Bold Italic, Bernard MT Condensed, and Inter. Every font uses font-display: swap for instant display while loading.

Font Size (Numeric Input)

Click the number field next to “Size” and type any number directly. You can enter values like 14, 24.5, 36, or 72. Press Tab or click elsewhere to apply. This precise input lets you set exact sizes without clicking repeatedly.

Bold & Italic

Click B to toggle bold weight. Click I to toggle italic style. Both can be active simultaneously.

Text Color

Click the colored square to open a full color picker. Choose any color using the color wheel, or type a hex code directly (e.g., #95201d for the brand red). The color applies immediately and is saved with your session.

Letter Spacing

Use the − and + buttons to decrease or increase the spacing between individual letters. A value of 0 is normal spacing. Positive values spread letters apart for an elegant headline look.

Line Height

Use the − and + buttons to adjust the spacing between lines of text. A value of 1.4 means line spacing is 1.4× the font size. Increase it for more breathing room in longer descriptions.

Text Alignment

Alignment buttons (Left, Center, Right) control how text is aligned within its element box. Center is common for titles and prices; Left is standard for descriptions.

Shape Design: When a shape is selected in the DESIGN tab, you will see Fill Color, Corner Radius, Border Width, and Border Color instead of typography controls.

14. The ARRANGE Tab – Position, Order & Rotation

The ARRANGE tab gives you precise numerical control over an element's position, size, and layer order. It also includes a rotation control.

ARRANGE Tab Controls
X: 142 Y: 87 W: 280 H: 40
Rotate: [====15°=========] ⇧ Fwd ⇩ Back ⇧⇧ Top ⇩⇩ Bottom

X and Y Position

The X and Y fields show the element's exact position in pixels measured from the top-left corner of the canvas. Type a new number to move the element to an exact coordinate. Useful for aligning multiple elements to the same X value.

Width and Height

Shows and sets the element's exact size in pixels. Type new values to resize precisely. For text elements, the width determines the text wrap boundary.

Rotation

Drag the rotation slider or type a degree value (0–360) to rotate the element around its center point. Useful for angled labels, watermarks, or decorative text at a slant.

Layer Order Controls

  • ⇧ Bring Forward: Moves the element one layer forward, in front of the next element above it
  • ⇩ Send Backward: Moves the element one layer back, behind the next element below it
  • ⇧⇧ Bring to Front: Moves the element to the very front, on top of everything else
  • ⇩⇩ Send to Back: Moves the element to the very back, behind everything else
Common use: When you add a rectangle behind text, click the rectangle, go to ARRANGE → Send to Back. The text always appears on top of the colored background shape.

17. Save, Load & Auto-Save

💾 Save Session

Saves your entire current state to the server. This includes every element's content, position, style, layer settings, visibility, lock state, and the positions of all custom images. You will see a green toast: “Project Saved Locally”.

📁 Load Session

Loads the latest server-saved version of your menu. Use this when continuing on a new device or browser. All unsaved local changes will be replaced by the server version. Always save before loading if you have unsaved work you want to keep.

Auto-Save on Export

Every time you click ⇩ Export Pro PNG, the editor automatically runs a Save first. This ensures your exported file always matches exactly what is saved on the server.

Local Browser Backup

In addition to the server save, the editor maintains a local backup in your browser's storage. This protects your most recent work even if your internet connection drops temporarily.

Best practice: Save after completing each menu section, before closing the browser, and before switching to another device. These three habits protect all your work.

18. Undo History

The undo system lets you reverse mistakes one step at a time. The editor remembers your last 30 actions in its history stack.

What Undo Covers

  • Text content changes (typing, deleting characters)
  • Font family, font size, color, bold/italic changes
  • Letter spacing and line height changes
  • Element moves (drag to new position)
  • Element additions (Add Text, Add Rect, image placement)
  • Element deletions
  • Visibility toggles
  • Lock toggles
  • Shape fill/border style changes
  • Opacity changes
  • Rotation changes
  • Layer order changes (bring forward, send back)

What Undo Does NOT Cover

  • Reset to Original — this is an intentional full restore, not undoable
  • Load Session — replaces the entire state from the server
  • Changes made before the 30-step history limit
Shortcut: Press Ctrl + Z on Windows/Linux, or Cmd + Z on Mac, to undo without clicking the button. Works at any time while the editor is open.

19. Export Pro PNG – Print-Ready Output

When your menu is complete, Export Pro PNG creates the final professional file for printing. This is the most important output of the entire editor.

What Export Produces

  • A PNG image file downloaded directly to your device
  • Resolution: 3600 × 5400 pixels
  • Print size: 12 inches × 18 inches at 300 DPI
  • All visible elements rendered at full quality
  • Hidden elements are excluded from the export automatically
  • Original full-resolution background is used (not the editor preview)

How to Export

  1. Open the drawer (🛠️ FAB button).
  2. Click ⬇️ Export Pro PNG.
  3. The editor auto-saves your session first (toast confirmation appears).
  4. A progress toast shows "Rendering export…".
  5. The file downloads automatically to your device\'s Downloads folder.
  6. A completion toast confirms the export is ready.
Keyboard shortcut: Press Ctrl + E (or Cmd + E on Mac) to trigger the export instantly from anywhere in the editor.
Before sending to print: Open the exported PNG and zoom in to verify all text is crisp and correctly placed. Confirm prices and names are accurate. Send the file to your print shop as-is — do not reduce its resolution.

23. Reset to Original

The 🔄 Reset to Original button restores your menu to its original factory template, removing all customizations. It is a complete reset — not a one-step undo.

How Reset Works

  1. Click 🔄 Reset to Original in the drawer.
  2. A confirmation dialog appears asking you to confirm the action.
  3. If you confirm, all custom text content, positions, styles, added elements, and uploaded image placements are cleared.
  4. The menu returns to the original template layout.
Warning: Reset to Original CANNOT be reversed with the Undo button. Before resetting, consider using 💾 Save Session to create a server backup of your current work. If you regret the reset, you can use 📂 Load Session to restore the last saved version.

When to Use Reset

Use Reset only when you want to start completely from scratch — for example, creating a seasonal menu from a clean template, or undoing extensive edits that went in the wrong direction. For single-step mistakes, use ↺ Undo Last Change instead.

24. Cross-Device Syncing

You can start editing on one device and continue on another. The syncing is powered by the server-based Save/Load system.

How to Continue Work on Another Device

  1. On your first device, complete your changes.
  2. Click 💾 Save Session. Wait for the green toast confirmation.
  3. Open your editor link on the second device.
  4. Click 📂 Load Session to load your saved work.
  5. Continue editing from exactly where you left off.
Do NOT skip the Save step. If you switch devices without saving, the second device will load the last saved version — which may be hours old. Always save before switching.

26. Troubleshooting

I cannot select or click any element

The layout may be Locked. Click the 🔒 Layout Locked button in the top header to switch to 🔓 Unlocked mode. Also confirm the element is not individually locked — check the Layers Panel and look for the 🔒 icon next to it.

I cannot move an element after selecting it

Either the global layout is Locked (header button), or the element itself is individually locked (LAYER tab or Layers Panel). Unlock both if needed. Also confirm you are clicking to select first, then dragging — do not drag without first confirming selection.

My text will not go into editing mode

Double-click (or double-tap on mobile) on the text element. A single click only selects it — you need a second click/tap to enter the text editor and place the cursor.

The Selection Bar is not visible

Make sure you have actually selected an element (single click). The bar only appears when something is selected. On mobile, scroll up on the canvas or close the drawer to make more room for the bar to appear.

I cannot see all the controls in the Selection Bar on my phone

The bar scrolls horizontally on mobile. Swipe left or right directly on the bar to see all tabs and buttons. The three tabs (LAYER, DESIGN, ARRANGE) and the duplicate and delete buttons should all be accessible by scrolling.

My changes are not showing on another device

You must click 💾 Save Session on the first device before using 📂 Load Session on the second. The editor does not sync automatically — only a manual Save sends the data to the server.

The background image looks blurry while editing

This is intentional. The editor uses a compressed preview version of the background for speed. Your exported PNG always uses the original full-resolution background and will be completely crisp and print-ready.

My export file is not downloading

Check your browser\'s download settings and make sure it allows downloads from the editor\'s site. On mobile, check that your browser has permission to save files. Try a different browser if the issue persists. Also check that you have enough storage space on your device.

The editor seems frozen or unresponsive

Save your work first (if possible), then click ↺ RELOAD in the top header to do a full page refresh. After reloading, click 📂 Load Session to restore your last saved version.

I accidentally deleted the wrong element

Immediately click ↺ Undo Last Change in the top header (or press Ctrl+Z). The deleted element will be restored as long as you have not exceeded the 30-step undo history.

An element ended up behind everything and I cannot reach it

Use the Layers Panel (in the drawer) to find the element by name, then click its row to select it. Once selected, go to the ARRANGE tab and click Bring to Front.

27. Best Practices

  • Save frequently. After finishing each menu section, after important changes, and always before closing the browser.
  • Keep the layout Locked by default. Only unlock when actively moving something. This is the single most effective habit for preventing accidents on both desktop and mobile.
  • Use Undo immediately when you notice a mistake — before doing anything else. The undo history clears older steps as you make new changes.
  • Zoom in on mobile before tapping small text like prices or fine-print descriptions. A zoomed-in tap is far more precise.
  • Name your elements meaningfully in the LAYER tab (e.g., "appetizer-section-title", "main-price-2"). This makes the Layers Panel much easier to navigate on complex menus.
  • Use the Layers Panel to select elements that are hard to click directly, hidden, or behind other elements.
  • Test your export before sending to print. Open the downloaded PNG file, zoom in to 100%, and verify every text element is correct and crisp.
  • Lock background elements. Any element that forms the permanent structure of your menu (borders, the background image, permanent logos) should be set to Locked so it can never be accidentally moved.
  • Use the ARRANGE tab for pixel-perfect alignment. If two prices need to be at exactly the same left edge, select each one and type the same X value.
  • Before resetting, always Save first so you can Load the last version if you change your mind.
