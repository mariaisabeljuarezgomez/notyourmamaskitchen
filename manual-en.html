1. Welcome

Welcome to Dine In Menu Editor Pro.

This editor allows you to update restaurant menu text, reposition items when needed, save your progress, continue on another device, and export a professional print-ready file.

It is designed to be understandable for everyday business use without requiring design software or advanced technical training.

What This Manual Covers

  • How to edit text and move items safely
  • How saving and loading work across devices
  • How the mobile version behaves, including toolbar scrolling, panning, pinch-to-zoom, and touch-safe layout controls
  • How to export a final professional PNG

2. Before You Start

Recommended Browser

  • Google Chrome

  • Apple Safari

Internet Connection

An active internet connection is required for permanent saving and cross-device syncing. Save Session stores your work to the server, and Load Session retrieves the latest saved version when continuing on another device. [file:1]

3. Quick Start Workflow

  1. Open the editor using your provided link.
  2. Open the main menu by tapping or clicking the floating action button (FAB).
  3. Turn Edit Mode ON.
  4. Keep Layout Locked enabled while editing text.
  5. Select a text item and make your changes.
  6. If you need to move an item, switch to Layout Unlocked, reposition the item, then immediately switch back to Layout Locked. [file:1]
  7. Click Save Session to store your work on the server. [file:1]
  8. Click Export Pro PNG when you are ready to create the final print-ready file. [file:1]

4. What Each Action Affects

Local Draft

As you work, the editor keeps a temporary draft in your current browser. This helps protect recent changes if the page closes unexpectedly, but it is not your permanent saved version.

Save Session

Save Session uploads your current work to the permanent server-backed system so it can be recovered later or loaded on another device. [file:1]

Load Session

Load Session replaces your current view with the latest version saved on the server. [file:1]

Undo Last Change

Undo Last Change reverses one recent action at a time, up to 50 steps, and covers text edits, drag moves, style changes, add text, and delete actions. [file:1]

Reset to Original

Reset to Original restores the original template and clears all custom edits. It is separate from Undo Last Change and should be used only when you want a full restart. [file:1]

Export Pro PNG

Export Pro PNG creates the final 12-inch, 300 DPI PNG file for professional printing or delivery. [file:1]

5. Main Controls Reference

Floating Action Button (FAB)

The floating action button is the main entry point to the control drawer. Use it to access editing mode, layout lock, save, load, undo, add text, export, and reset.

Edit Mode

Turn Edit Mode ON when you want to change text such as prices, dish names, descriptions, or titles.

Layout Locked / Layout Unlocked

Layout Locked is the default state each time the editor loads. Use it for normal text editing, panning, zooming, and safe navigation. Use Layout Unlocked only when you want to physically drag an item to a new position. [file:1]

Save Session

Use Save Session after important changes, before closing the editor, and before switching devices. [file:1]

Load Session

Use Load Session to retrieve the latest saved version from the server when continuing on another device. [file:1]

Undo Last Change

Undo Last Change reverses one action at a time and covers text edits, moved items, style changes, added text, and deleted items. [file:1]

Add Text

Add Text creates a new blank text element at the center of your current view. [file:1]

Export Pro PNG

Export Pro PNG creates the final high-resolution PNG using the original high-resolution source rather than the editing preview. [file:1]

Reset to Original

Reset to Original clears all custom changes and restores the original template. This action is separate from Undo and cannot be reversed by Undo Last Change. [file:1]

6. Selection Toolbar

The selection toolbar appears when a text item is selected.

  • Font Dropdown for changing the font
  • Size (+ / -) for changing text size
  • Color Picker for changing text color
  • Letter Spacing for adjusting spacing between letters
  • Delete Element for removing the selected item

Mobile Toolbar Behavior

On mobile devices, the toolbar appears near the bottom of the screen and may be horizontally scrollable because phone screens are narrower. If you do not see all tools right away, swipe left or right on the toolbar itself to reveal the remaining controls.

Deleting an item requires confirmation, and the item can be restored with Undo Last Change as long as it remains within the undo history. [file:1]

7. Edit Mode vs. Layout Lock

FeatureControlsBest For
Edit ModeContent such as words and pricesChanging text, fixing typos, updating prices
Layout LockPosition on the pageMoving text blocks, headings, or price alignment

Use Edit Mode when you want to change words. Use Layout Unlocked only when you want to move something. Return to Layout Locked as soon as the item is in place. [file:1]

This separation is especially important on mobile, where touch gestures can otherwise cause accidental movement. [file:1]

8. Detailed Editing Behavior

Selecting and Editing Text

  1. Ensure Edit Mode is ON.
  2. Tap or click an item once to select it.
  3. A highlight box and the selection toolbar will appear.
  4. Tap the text again, or double-tap/double-click depending on your device, to place the cursor.
  5. Type your changes.
  6. Tap the background or select another item to finish editing.

Moving Items

  1. Open the main menu.
  2. Switch to Layout Unlocked.
  3. Select the item you want to move.
  4. Drag it to the new position.
  5. Switch back to Layout Locked immediately after placement. [file:1]

9. Cross-Device Syncing

  1. Finish your edits on the first device.
  2. Click Save Session.
  3. Open the same editor link on the second device.
  4. Click Load Session.

If you forget to save on the first device, the second device will load the last saved version rather than your newest unsaved changes. [file:1]

10. Mobile Guide

The mobile version is designed for safe editing on phones and tablets. Because touch input is less precise than a mouse, the workflow protects the layout first and only allows movement when you intentionally unlock it. [file:1]

How the Mobile Version Works

The layout opens locked by default so you can pan, zoom, and inspect the menu without accidentally dragging text. [file:1]

Main Mobile Areas

  • FAB: Opens the main drawer with Edit Mode, Layout Lock, Save Session, Load Session, Undo Last Change, Add Text, Export Pro PNG, and Reset to Original.
  • Canvas Area: The main visual menu area used for panning, zooming, selecting text, and reviewing spacing.
  • Bottom Selection Toolbar: Appears after selecting text and may be horizontally scrollable on phones.
  • Floating Zoom Controls: When Layout Unlocked is active, floating + and - zoom controls should appear for more precise control. [file:1]

Mobile Gestures

  • Pan: Use one finger to move around the canvas.
  • Pinch to Zoom: Use two fingers to zoom in or out.
  • Select: Tap once to select a text item.
  • Edit: Tap again after selection to place the cursor and show the keyboard.
  • Drag to Move: Only while Layout Unlocked is enabled.

Multi-Touch Protection

If a multi-touch gesture begins while dragging, the drag should be cancelled or suppressed immediately so the layout is not shifted by accident during pinch zoom. [file:1]

Best mobile workflow: Keep Layout Locked on while navigating and editing text. Unlock only when repositioning, then lock it again immediately. [file:1]

11. Troubleshooting

I can type text but I cannot move items

Make sure Layout Unlocked is enabled. The editor opens in Layout Locked by default. [file:1]

I can move items but I cannot edit text

Make sure Edit Mode is ON.

I cannot see all toolbar buttons on my phone

Swipe left or right on the bottom toolbar. On mobile, it may be horizontally scrollable so all tools can fit on screen.

My changes are not showing on another device

Make sure you clicked Save Session on the first device before using Load Session on the second device. [file:1]

The editor preview looks lower quality

This is normal. The editor uses a lightweight preview for speed, while Export Pro PNG uses the original high-resolution source for final output. [file:1]

12. Best Practices

  • Keep Layout Locked on unless you are actively moving items.
  • Save after each important section of work.
  • Use Undo Last Change immediately when you notice a mistake. [file:1]
  • Review your exported PNG before printing.
  • On mobile, zoom in before selecting small prices or fine text.
  • Swipe the toolbar on mobile if not all controls are visible.
  • Re-lock the layout immediately after moving any item. [file:1]
