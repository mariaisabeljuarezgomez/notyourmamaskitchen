2. Before You Start

Recommended Browsers

  • Google Chrome — Best overall experience on desktop and Android
  • Microsoft Edge — Excellent on Windows desktop
  • Apple Safari — Best on iPhone and iPad
  • Mozilla Firefox — Fully supported on desktop

Internet Connection

An active internet connection is required for saving to the server and loading previously saved work on another device. The editor also uses your browser's local storage as a backup so your most recent work is protected even if the connection drops temporarily.

Screen Size & Orientation

The editor works on any screen size. On desktop you see the full canvas and all controls simultaneously. On phones, the canvas fills the screen and controls appear as a floating toolbar. On phones and tablets, landscape orientation gives more working space and makes it easier to see the full toolbar.

Preview Quality vs. Export Quality

While editing, the background image displays at a lower resolution so the editor loads fast and stays responsive. This is normal and intentional. When you click Export Pro PNG, the editor uses the original full-resolution background file, producing a crisp professional 300 DPI print file regardless of how the preview looked.

Best habit: Save your session regularly — after completing a section of work, before closing the browser, and before switching to another device. One click on Save protects everything.

Your Editor Link

Your editor is accessed through a private web address provided to you by MARIAS DIGITAL DESIGNS. Bookmark it in your browser for quick return. Never share this link publicly — it is your private editing environment.

Tip: On iPhone or Android, save your editor link to your home screen so it opens like an app. In Safari or Chrome, use the browser menu and select "Add to Home Screen."

4. Quick Start Workflow

Follow these steps for your first editing session. Each step is explained fully in later sections.

1

Open your editor link in Google Chrome or Safari. The menu loads automatically with your last saved session.

2

Unlock the layout by clicking the 🔒 Layout Locked button in the top header. It turns yellow and shows 🔓 Layout Unlocked. This lets you click and select elements.

3

Click any text element on the canvas to select it. A yellow highlight box appears around it and the Selection Bar appears at the bottom.

4

Double-click the text (or double-tap on mobile) to place your cursor inside it. Type your changes — update a price, fix a spelling, change a dish name.

5

Click the canvas background (not on any element) to deselect and finish editing.

6

Use the DESIGN tab in the Selection Bar to change the font, color, size, line height, or letter spacing.

7

Click 💾 Save in the top header to save your work to the server. A green toast notification confirms the save.

8

When ready to print, open the drawer (🛠️) and click ⬇️ Export Pro PNG. Your 300 DPI file downloads automatically.

Remember: Keep the layout Locked whenever you are editing text or browsing. Unlock it only when dragging something to a new position, then lock it again immediately.

5. The Top Header Bar

The top header bar is always visible at the top of the screen and contains four controls you will use constantly.

Top Header Bar
MENU EDITOR PRO V2 💾 Save ↺ Undo Last Change 🔒 Layout Locked ↺ RELOAD

💾 Save

Saves your entire session — all text content, positions, styles, layer settings, and image placements — to the server. Also creates a local browser backup simultaneously. A green toast message confirms: "Project Saved Locally". Use Save frequently. There is no automatic timed saving.

Shortcut: Save runs automatically every time you click Export Pro PNG, so your export always matches your most recent work.

↺ Undo Last Change

Reverses your most recent action one step at a time. The undo system remembers up to 30 actions, covering text edits, element moves, style changes, additions, deletions, visibility changes, and lock toggles. Click repeatedly to step back through history.

Important: Undo does NOT reverse a Reset to Original or a Load Session. Those are intentional full-restore actions.

🔒 Layout Locked / 🔓 Layout Unlocked

Two states:

  • 🔒 Layout Locked (red, default): Elements cannot be accidentally dragged. Use for all text editing, styling, and general navigation.
  • 🔓 Layout Unlocked (yellow): Elements CAN be dragged to new positions. Floating Zoom Controls (+/−) also appear in the top-right corner. Use only when repositioning something, then immediately switch back to Locked.
Rule of thumb: If you are not actively dragging something, the layout should be Locked. This prevents accidental repositioning, especially on mobile.

↺ RELOAD

Refreshes the entire page (hard browser reload). Use only if the editor becomes unresponsive or appears visually broken. Save your work before clicking RELOAD — any unsaved changes will be lost.

7. The Asset Tray – Built-In & Uploaded Images

Inside the Tools Drawer, at the very top, is the Asset Tray — a horizontal scrolling row of image thumbnails. These are your menu's built-in graphic assets plus any images you have personally uploaded.

Asset Tray — Top of the Drawer
🖼️ 🖼️ 🖼️ ← Scroll left/right to see all

Using Built-In Images

The built-in images are pre-loaded decorative graphics designed to match your menu's visual style. To add one to your canvas, simply click or tap its thumbnail in the tray. The image is placed on the canvas and selected automatically so you can reposition and resize it immediately.

Uploading Your Own Images

  1. Click 🖼️ Upload Img in the drawer.
  2. Your device's file picker opens. Select a JPG, PNG, WEBP, GIF, or SVG file.
  3. The image uploads, is stored in your permanent library, appears in the Asset Tray, and is placed on the canvas automatically.
  4. Use the resize handles (corner circles visible when selected) to adjust its size, then drag to position.
Tip: Uploaded images persist in your library permanently. Even after clearing the canvas, your uploaded images will still be in the Asset Tray ready to use again.

Transparent PNG Images

If you upload a PNG file with a transparent background (such as a logo), the editor automatically trims empty transparent space before placing it — no white box behind your logo.

Image File Tips

  • Use PNG for logos and graphics with transparent backgrounds
  • Use JPG for photographs of food or your restaurant
  • For best print quality, upload images at 300 DPI or higher resolution
  • Most standard phone photos upload without issue

8. Adding & Managing Text Elements

Adding New Text

  1. Open the drawer (🛠️ FAB button).
  2. Click ＋ Add Text.
  3. A new element labeled "New Text" appears at the center of your current view, already selected.
  4. Double-click (or double-tap on mobile) to enter editing mode and type your content.
  5. Click anywhere on the canvas background to finish editing.

Selecting an Existing Text Element

Make sure layout is unlocked (🔓), then click or tap once on any text element. A yellow glowing border appears around it and the Selection Bar shows at the bottom. The element is now selected.

Editing Text Content

With a text element selected, double-click it (or double-tap on mobile) to place your cursor inside. Type your changes — update a price, fix a name, change a description. Press Enter for a new line. Click the canvas background to confirm when done.

Tip: Text supports multiple lines. Press Enter/Return while editing to create a line break inside the same text element.

Moving Text Elements

  1. Ensure layout is 🔓 Unlocked.
  2. Click the element once to select it (do NOT double-click — that enters text editing mode).
  3. Hold and drag it to its new position.
  4. Release. The new position is recorded in undo history.
  5. Lock the layout again with 🔒 once finished.

Deleting a Text Element

Select the element, then click the 🗑 Delete button (red trash icon) in the Selection Bar. A confirmation dialog appears. Confirm deletion. You can restore it with ↺ Undo Last Change if needed.

Duplicating an Element

Select any element. In the Selection Bar, click the 📋 Duplicate button (blue). An exact copy appears, offset slightly from the original, and is automatically selected for immediate repositioning. Works for text, images, and shapes.

Background-Labeled Elements

If a text or image element is named "background" and set to Locked in the LAYER tab, it will not respond to click selection while the global layout is locked — protecting the base design from any accidental interaction.

9. Adding & Managing Image Elements

Adding an Image from the Asset Tray

Open the drawer and click any image thumbnail in the Asset Tray. The image is placed on the canvas at a default size and is automatically selected. Drag it to position, and use the corner resize handles to change its size.

Resizing Images

When an image element is selected, you will see four yellow circular handles at each corner.

  • Drag any corner handle to resize the image
  • Resize is zoom-aware — handles work correctly whether zoomed in at 200% or out at 50%
  • Minimum size protection prevents images from being shrunk accidentally to nothing
  • Hold Shift while dragging a corner to resize proportionally (keeps original aspect ratio)

Moving Images

Ensure layout is 🔓 Unlocked, click the image to select it, then drag it by its body (not by a corner handle — those resize). Release to confirm the new position.

Image Opacity

Select an image element, go to the LAYER tab, and use the Opacity slider to fade it. Useful for watermarks, decorative backgrounds, or semi-transparent overlays. Range: 0% (invisible) to 100% (fully solid).

Hiding an Image Without Deleting

In the LAYER tab, click the 👁️ Visibility toggle to hide the image. It becomes faint (20% opacity) in the editor but will not appear in your exported PNG. The hidden element still shows in the Layers Panel so you know it exists.

Locking an Image in Place

In the LAYER tab, click the 🔒 Lock Element toggle. A locked element cannot be moved, resized, or accidentally selected. Ideal for permanent background frames or watermarks.

Background images: Any element named "background" AND set to Locked will not respond to click selection while the global layout is also locked — giving you double protection on critical base design elements.

12. The LAYER Tab

The LAYER tab appears first in the Selection Bar when you select any element. It controls visibility, lock state, transparency, name, and role.

LAYER Tab Controls
LAYER 👁️ Visible 🔓 Unlocked Opacity: [========80%===]
Name: [price-header_______] Role: Content ▼

👁️ Visibility Toggle

Toggles the element between visible and hidden. Hidden elements appear very faint (🙈 at 20% opacity) in the editor so you can still see their position, but they are completely invisible in the exported PNG. Hidden elements show with a strikethrough in the Layers Panel.

🔒 Lock Element Toggle

When locked, the element cannot be selected, moved, or resized. Hover shows a locked cursor. To unlock an element, select it from the Layers Panel and turn off the lock. Essential for background frames and permanently anchored elements.

Opacity Slider

Drag left to make the element more transparent, right to make it fully opaque. Range: 0% to 100%. Changes are visible immediately and recorded in undo history.

Element Name

A label used to identify the element in the Layers Panel. Default names are "text-1", "image-1", etc. Type a meaningful name like "main-title" or "price-left" for easier organization. The name never appears on the printed menu.

Role Selector

  • Content (default): A standard element participating normally in the design.
  • Background: Treated as a background layer. If also Locked, it will not respond to click selection when global layout is locked — fully protecting your base design.
  • Overlay: Always rendered above Content elements in both the editor and the final export.

13. The DESIGN Tab – Fonts, Color, Size & Spacing

The DESIGN tab contains all styling controls for the selected element. Typography controls appear for text elements; shape controls appear for rectangles.

DESIGN Tab — Text Element
Font: Century Gothic ▼ Size: 24 B I
Color: Spacing: − 0.05em + Line H: − 1.4 +

Font Family Selector

Click the font dropdown to see all available fonts. The editor includes premium fonts: Century Gothic, Century Gothic Bold, Century Gothic Bold Italic, Bernard MT Condensed, and Inter. All fonts use fast-loading font-display:swap for instant display.

Font Size — Numeric Input

Click the number field and type any value directly — 14, 24.5, 36, 72. Press Tab or click elsewhere to apply. Enter the exact size you need without stepping through values one by one.

Bold & Italic

Click B to toggle bold. Click I to toggle italic. Both can be active simultaneously. Fonts with dedicated bold-italic variants (like Century Gothic Bold Italic) override the style toggle automatically when selected.

Text Color

Click the colored square to open a full color picker. Choose from the color wheel or type a hex code directly (e.g., #95201d for brand red, #f1c40f for gold). Color applies immediately and is saved with your session.

Letter Spacing

Use − and + to decrease or increase spacing between individual letters. Measured in em units relative to font size. Positive values spread letters apart for elegant headline styles. Negative values tighten them.

Line Height

Use − and + to adjust spacing between lines of text. A multiplier of 1.4 means line spacing is 1.4× the font size. Increase for more breathing room in longer descriptions.

Text Alignment

Left, Center, Right alignment buttons control how text sits within its element box. Center is standard for titles and prices. Left is standard for descriptions.

Shape DESIGN controls: When a shape is selected, the DESIGN tab shows Fill Color, Corner Radius, Border Width, and Border Color instead of typography controls.

14. The ARRANGE Tab – Position, Order & Rotation

The ARRANGE tab gives you precise numerical control over an element's position, size, rotation, and layer order.

ARRANGE Tab Controls
X: 142 Y: 87 W: 280 H: 40

X and Y Position

Shows the element's exact pixel position from the top-left corner of the canvas. Click and type a new number to move to an exact coordinate. Useful for aligning multiple elements — set all price labels to the same X value for a perfect column.

Width and Height

Shows and sets the element's exact pixel size. For text elements, width determines the text wrap boundary — wider means text stays on fewer lines.

Rotation

Drag the slider or type a degree value (0–360°) to rotate the element around its center. 0° is upright, 90° tilts right, 180° is upside-down, 270° tilts left. Use for angled labels, diagonal watermarks, or decorative tilted text.

  • ⬇ Send Backward: Moves the element one layer back (behind the next element)
  • ⬆⬆ Bring to Front: Moves the element to the very top, in front of everything
  • ⬇⬇ Send to Back: Moves the element to the very bottom, behind everything
Common use: Add a rectangle, then click ARRANGE → Send to Back. Your text automatically appears on top of the colored shape.

17. Save, Load & Auto-Save

💾 Save Session

Saves your entire state to the server: every element's content, position, style, layer settings, visibility, lock state, and image placements. Creates both a server backup and a local browser backup simultaneously. Confirmation toast: "Project Saved Locally".

📂 Load Session

Loads the latest server-saved version of your menu. Use when continuing on a new device or browser. All unsaved local changes will be replaced by the server version. Always save first if you have unsaved work.

Auto-Save on Export

Every time you click Export Pro PNG, the editor automatically runs a Save first. Your exported file always matches exactly what is on the server. A brief toast confirms the auto-save before export begins.

Local Browser Backup

The editor maintains a local backup in your browser's storage. This protects your most recent work if your internet connection drops temporarily. Updated every time you Save.

Best practice: Save after completing each menu section. Before closing the browser. Before switching to another device.

18. Undo History

The ↺ Undo Last Change button (top header bar, or press Ctrl+Z) steps back through your editing history one action at a time. The undo system tracks up to 30 actions.

What Undo Covers

  • Text content edits
  • Element drag moves and position changes
  • Style changes (font, color, size, spacing)
  • Adding new elements (text, image, shape)
  • Deleting elements
  • Visibility and lock state changes
  • Opacity and rotation changes
Undo does NOT cover: Reset to Original, Load Session, or browser page reload. These are intentional full-restore actions that cannot be undone.

19. Export Pro PNG – Print-Ready Output

The ⬇️ Export Pro PNG button in the Tools Drawer produces a professional, print-ready PNG file. This is the only way to get a file suitable for commercial printing or sending to a print shop.

Export Button — Inside the Drawer
⬇️ Export Pro PNG ← Also press Ctrl+E on desktop

What Happens During Export

  1. Auto-Save runs first. Your current session is saved to the server before anything else. This guarantees your printed file matches what is saved.
  2. The master background image loads. The editor swaps in the high-resolution master background file (menu_bg.png) in place of the fast preview image used during editing.
  3. The canvas renders to PNG. Every visible element — text, images, shapes — is composited at full resolution.
  4. 300 DPI metadata is injected. The PNG binary is rewritten to embed proper physical density information (pHYs chunk) so printers and design software correctly recognize the file as 300 DPI.
  5. The file downloads automatically to your device's Downloads folder.

Export Output Specifications

File Format

  • Specify that the file is 300 DPI, RGB color
  • Request a proof if possible before a large print run
  • For very large format prints, ask the printer if they require a higher resolution source

26. Troubleshooting

This section covers the most common issues you may encounter and how to resolve each one quickly.

I cannot click on an element

Most likely cause: The layout is Locked. Click the 🔒 Layout Locked button to switch to 🔓 Unlocked mode, then try clicking the element again.

Second possible cause: The element has its Lock toggle turned on in the LAYER tab. Open the Layers Panel in the drawer, find the element's row, and click its 🔒 icon to unlock it.

I accidentally moved an element to the wrong position

Click ↺ Undo Last Change in the top header bar (or press Ctrl+Z) immediately. The element will return to its previous position. The undo system tracks up to 30 steps, so you can step back through several accidental changes if needed.

My text edits are not showing up

Make sure you double-clicked the text element to enter editing mode (you should see a blinking cursor). Single-clicking only selects the element without entering edit mode. Also confirm you clicked the canvas background to commit your changes after typing — text edits are confirmed when you exit the text element.

The exported PNG looks blurry or low quality

The exported file is always generated at full 300 DPI resolution regardless of how it looks on screen. If the image appears soft when viewing on your computer, try opening it in a photo viewer that displays at 100% zoom. Very large PNG files at 300 DPI appear small on screen because the screen has fewer pixels per inch than the file. Send the file directly to your printer — it will print crisply.

A hidden element is appearing in my export

Check the element's visibility in the Layers Panel. The 👁️ icon must be toggled to 🙈 (hidden) for the element to be excluded from export. If the icon shows as 👁️, the element is visible and will appear in the export.

The editor is running slowly or appears frozen

Try clicking the ↺ RELOAD button in the top header bar. Save your work first before reloading — any unsaved changes will be lost. After reloading, use Load Session to restore your last saved state.

I lost my changes after closing the browser

Changes are only permanently saved when you click the 💾 Save button. The editor does maintain a local browser backup, but this can be cleared by browser updates or cache cleaning. Always click Save before closing the browser. Best habit: Save after every significant edit.

Save or Load is failing or hanging

This usually indicates a temporary internet connection issue. Check your connection. The editor will retry automatically. If the problem persists, try the RELOAD button, then attempt Save again. The local browser backup protects your most recent work during connection interruptions.

An image I uploaded is not appearing on the canvas

Check the Layers Panel — the image may have been placed but is hidden (🙈) or has Opacity at 0%. Also check that your image file is a supported format: JPG, PNG, WEBP, GIF, or SVG. Very large files may take a moment to process — wait a few seconds and check again.

I cannot find the Selection Bar — it disappeared

The Selection Bar only appears when an element is selected. Click or tap any element on the canvas to bring it back. If you previously dragged it off-screen, try reloading the page (save first) to reset the toolbar position.

27. Best Practices

These habits will protect your work, keep your menu looking professional, and make editing fast and frustration-free.

Saving Habits

  • Save after every section of work. Update a group of prices → Save. Add a new item description → Save. Move an image into position → Save.
  • Save before closing the browser — even if you are "just stepping away for a minute."
  • Save before switching devices. Server-side save is the bridge between your phone and your computer.
  • The 💾 Save button is in the top header bar — always one tap away.

Layout Lock Habits

  • Keep Layout Locked by default. Only unlock when you need to drag something. Re-lock immediately after.
  • On mobile especially: Locked mode prevents dozens of potential accidental repositioning errors.
  • Text editing (double-click to type) works in both Locked and Unlocked mode — you never need to unlock just to edit text.

Typography Best Practices

  • Keep section headers larger (28–36px) and item names smaller (16–22px) for clear visual hierarchy.
  • Use Center alignment for titles, prices, and featured items. Use Left alignment for descriptions and lists.
  • Limit your menu to 2–3 font choices maximum for a clean, professional look.
  • Use Letter Spacing (positive values) on headers for an elegant, premium appearance.
  • Increase Line Height (1.4–1.6) for paragraph descriptions to improve readability.

Image Best Practices

  • Use PNG with transparency for logos — the editor auto-trims the transparent edges.
  • Upload images at the highest resolution you have — the editor handles large files well.
  • Use the Opacity slider in the LAYER tab to fade decorative images that should not compete with text.
  • Lock any image that should never move (background frame, watermark) using the 🔒 Lock toggle.

Layer Organization

  • Give your elements meaningful names in the LAYER tab ("main-title", "price-col-1", "logo") for easy identification in the Layers Panel.
  • Use the Layers Panel to select hidden or stacked elements instead of trying to click them on the canvas.
  • Set background frames and decorative borders to Role: Background + Locked so they can never be accidentally moved.

Before Exporting

  • Review all text for spelling errors — zoom in on each section.
  • Check that all prices are correctly updated.
  • Verify that no elements you want visible are accidentally hidden (check the Layers Panel).
  • Save your session — Export auto-saves, but it is good habit to save manually first.
  • Export, then open the downloaded file and zoom to 100% to inspect quality before sending to print.
